### Install: 
```bash
git clone "https://github.com/n3krodamus/RepoStarter"
cd RepoStarter

Edit env_create and put your TOKEN, USERNAME and working directory

cat to_bashrc >> ~/.bashrc
source ~/.bashrc 

```

### Usage:
```bash
To run the script type in 'create <name of your folder>'
```
